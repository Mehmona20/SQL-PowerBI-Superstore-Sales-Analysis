# Superstore Sales Analysis Dashboard

This project analyzes Superstore sales data using SQL and Power BI.

## Features

* KPI Cards for Total Sales, Profit, and Quantity
* Sales and Profit Trend Analysis
* Category and Sub-Category Sales Analysis
* Regional Map Visualization
* Customer Segment Analysis
* Interactive slicers for Region, Category, and Segment

## Tools Used

* SQL
* Power BI
* DAX
* Power Query
